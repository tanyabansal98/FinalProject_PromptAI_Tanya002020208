# Bloom's Level: Understand
## What it means
Explaining ideas, summarizing, comparing, interpreting meaning.
## Question stems
- Explain... Describe... Compare... Contrast... Summarize... Interpret...
## Assessment types
- Short answer explanations
- Compare/contrast questions
- Explain in your own words
## Example
Q: Explain the difference between supervised and unsupervised learning.
A: Supervised learning uses labeled data to learn input-output mappings, while unsupervised learning finds hidden patterns without labels.
