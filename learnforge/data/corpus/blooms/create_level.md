# Bloom's Level: Create
## What it means
Designing new solutions, synthesizing ideas, producing original work.
## Question stems
- Design... Build... Propose... Create... How would you architect...
## Assessment types
- Design a system
- Write a proposal
- Build a prototype
## Example
Q: Design a recommendation system for a bookstore.
A: Use collaborative filtering on purchase history + content-based filtering on book metadata. Hybrid approach with A/B testing.
