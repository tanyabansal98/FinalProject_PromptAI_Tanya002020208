# Bloom's Level: Apply
## What it means
Using knowledge in new situations, solving problems, implementing.
## Question stems
- How would you use...? Implement... Solve... Apply... Calculate...
## Assessment types
- Problem-solving scenarios
- Code implementation tasks
- Case study applications
## Example
Q: Given a dataset of house prices, which algorithm would you choose and why?
A: Linear regression, because the target is continuous and the relationships appear linear.
