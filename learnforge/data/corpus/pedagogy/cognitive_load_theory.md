# Cognitive Load Theory
## What it is
Working memory has limited capacity. Learning fails when cognitive load exceeds this capacity. Design instruction to manage intrinsic, extraneous, and germane load.
## Types of load
- **Intrinsic**: Complexity of the material itself (can't reduce, but can sequence)
- **Extraneous**: Poor design that wastes cognitive resources (reduce this)
- **Germane**: Mental effort dedicated to learning (maximize this)
## Design principles
- Break complex topics into small chunks
- Use clear formatting and consistent structure
- Remove unnecessary decorative elements
- Use worked examples to reduce intrinsic load
- Build on prior knowledge to leverage schemas
