# Constructive Alignment
## What it is
A pedagogical framework where learning objectives, teaching activities, and assessment are all aligned. If you want students to 'analyze', your quiz must test analysis, not just recall.
## Components
1. **Intended Learning Outcomes (ILOs)** — What should the student be able to do?
2. **Teaching/Learning Activities** — What activities enable achieving the ILOs?
3. **Assessment** — How do you measure if ILOs were achieved?
## Application in content generation
- Map each module to specific Bloom's levels
- Generate quizzes that match the module's Bloom's level
- Provide practice activities at the right cognitive level
- Use rubrics that assess the intended outcomes
