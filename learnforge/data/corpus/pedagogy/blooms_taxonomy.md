# Bloom's Taxonomy
## Overview
A hierarchical framework for classifying educational learning objectives into levels of complexity:
1. **Remember** — Recall facts, definitions, lists
2. **Understand** — Explain concepts, summarize, compare
3. **Apply** — Use knowledge in new situations, solve problems
4. **Analyze** — Break down information, identify patterns and relationships
5. **Evaluate** — Make judgments, defend positions, critique
6. **Create** — Design new solutions, synthesize ideas, produce original work
## How to use in assessment
- Beginner learners: focus on Remember + Understand
- Intermediate: emphasize Apply + Analyze
- Advanced: target Evaluate + Create
- Mix levels in quizzes for comprehensive assessment
## Common pitfalls
- Staying at Remember level too long (just memorizing facts)
- Jumping to Create without building lower levels first
- Not explicitly telling learners what level they're operating at
