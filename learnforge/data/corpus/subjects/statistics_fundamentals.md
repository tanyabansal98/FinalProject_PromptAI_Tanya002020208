# Statistics Fundamentals
## Core concepts
Mean, median, mode. Standard deviation and variance. Normal distribution. Probability. Hypothesis testing (p-values, significance). Correlation vs causation.
## Common misconceptions
- Correlation does not imply causation
- p < 0.05 doesn't mean 95% probability the hypothesis is true
- Average can be misleading (use median for skewed data)
- Statistical significance ≠ practical significance
