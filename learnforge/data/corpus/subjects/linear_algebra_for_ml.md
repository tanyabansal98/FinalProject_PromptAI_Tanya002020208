# Linear Algebra for ML
## Core concepts
Vectors, matrices, dot products, matrix multiplication, transpose, inverse. Eigenvalues and eigenvectors. Dimensionality reduction (PCA). Matrix factorization.
## Why it matters for ML
- Data is represented as matrices (rows=samples, columns=features)
- Neural network operations are matrix multiplications
- PCA uses eigendecomposition for dimensionality reduction
## Common misconceptions
- Matrix multiplication is not element-wise (use @ not * in numpy)
- Inverse doesn't always exist (singular matrices)
