# Neural Networks Basics
## Core concepts
Layers of connected nodes (neurons). Each connection has a weight. Forward pass: input→hidden→output. Backpropagation adjusts weights to minimize loss. Activation functions (ReLU, sigmoid) add non-linearity.
## Architecture types
- Feedforward: simple, no loops
- CNN: spatial data (images)
- RNN/LSTM: sequential data (text, time series)
- Transformer: attention-based (GPT, BERT)
## Common misconceptions
- Neural networks don't 'think'
- Deeper ≠ always better (diminishing returns, training difficulty)
- CNNs aren't just for images
