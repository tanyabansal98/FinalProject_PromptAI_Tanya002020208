# Database Fundamentals
## Core concepts
Relational (SQL): tables, rows, columns, joins, ACID properties. NoSQL: document (MongoDB), key-value (Redis), column (Cassandra), graph (Neo4j). Indexing, normalization, transactions.
## Common misconceptions
- NoSQL doesn't mean 'no SQL' — it means 'not only SQL'
- Normalization isn't always better (denormalize for read performance)
- Indexes speed reads but slow writes
