# Data Structures Overview
## Core structures
Arrays (O(1) access), linked lists (O(1) insert), hash maps (O(1) lookup), trees (O(log n) search), graphs (relationships), stacks (LIFO), queues (FIFO).
## When to use what
- Need fast lookup by key → hash map
- Need sorted data with fast insert → balanced BST
- Need FIFO processing → queue
- Need to model relationships → graph
