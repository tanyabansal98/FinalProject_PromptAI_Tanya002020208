# Cloud Computing Introduction
## Core concepts
IaaS (VMs), PaaS (managed platforms), SaaS (applications). Major providers: AWS, Azure, GCP. Key services: compute (EC2/VMs), storage (S3/Blob), databases (RDS/CosmosDB), serverless (Lambda/Functions).
## Common misconceptions
- Cloud isn't just 'someone else's computer' — it's elastic, on-demand infrastructure
- Cloud isn't always cheaper (depends on usage patterns)
- Serverless still has servers — you just don't manage them
