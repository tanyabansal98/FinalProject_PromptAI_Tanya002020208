# Python Programming Basics
## Core concepts
Variables, data types (int, float, str, list, dict, tuple, set), control flow (if/elif/else, for, while), functions (def, args, kwargs, return), classes (OOP basics).
## Common misconceptions
- Python lists aren't arrays (they're dynamic arrays)
- = is assignment, == is comparison
- Mutable default arguments are shared across calls
- Indentation is not optional — it defines code blocks
