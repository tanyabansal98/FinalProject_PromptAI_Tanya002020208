# Web Development Misconceptions
## HTML is a programming language
HTML is a markup language — it structures content but doesn't have logic, loops, or variables.
## JavaScript and Java are related
They share a name for marketing reasons from the 1990s. They are completely different languages.
## HTTPS is just for e-commerce
All websites should use HTTPS. It protects against eavesdropping, tampering, and impersonation.
