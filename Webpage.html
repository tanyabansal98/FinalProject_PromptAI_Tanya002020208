What is LearnForge?

AI that builds entire courses for you — in seconds

The problem

Learning a new topic means hunting for tutorials, taking notes, quizzing yourself, and tracking what you don't know. There's no single tool that does all of this — personalized to your level, with real feedback, and for any topic.

The solution

Type a topic. Pick your level. LearnForge generates a structured curriculum, detailed lessons, quizzes with instant scoring, flashcards, and tracks your progress — all adapted to your demonstrated performance. Three AI agents work together to make it happen.

How simple is it?

STEP 1

Clone the repo, install dependencies, build the RAG index. Three commands.

pip install -e . && make index && make demo
STEP 2

Open the app, type any topic, and start learning. Demo mode works instantly with zero API keys.

STEP 3

Switch to API mode for real-time generation, upload your own study materials, or use Ollama for fully offline learning.

Features

Everything a learner needs,
generated by AI

Enter any topic. Pick your level. LearnForge builds a complete personalized learning experience.

📚

Structured curricula

Modules with Bloom's Taxonomy progression — from recall to synthesis. Visual progression chart shows cognitive complexity escalating across modules.

🧪

Interactive quizzes

MCQ, true/false, and short answer. Each question tagged with Bloom's level. Instant scoring with misconception detection and RAG-grounded feedback.

🃏

Spaced-repetition flashcards

Flip cards with hints, confidence rating, and mastery tracking. Optimized for long-term retention.

📊

Progress dashboard

Bloom's radar chart, score trends, knowledge gap heatmap, and study streaks. All powered by Plotly visualizations.

🎨

Multimodal content

Auto-generated concept diagrams (DALL-E / SVG) and text-to-speech audio narration on every lesson. Two modalities beyond text.

📁

Upload your own materials

Drag PDFs, TXT, or markdown files into the Data Manager. They get ingested into the RAG corpus so the assessor can cite your notes.

Core components

5 generative AI components implemented

LearnForge implements 5 core generative AI techniques in a single cohesive system.

01

Prompt engineering

5 system prompts with JSON schemas, Bloom's constraints, citation requirements

02

Fine-tuning

QLoRA on Qwen2.5-3B, 180 examples, eval loss 0.609 → 0.234

03

RAG

30-doc corpus, ChromaDB + BM25 hybrid, cross-encoder reranking

04

Multimodal

DALL-E image gen + OpenAI TTS audio narration on lessons

05

Synthetic data

180 instruction-tuned examples across 20 topics × 3 levels × 3 types
